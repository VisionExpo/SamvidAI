# SamvidAI Demo

## Demo Flow
1. Upload a contract PDF
2. Ask a legal question
3. SamvidAI retrieves relevant clauses
4. AI explains risks with references
5. Human reviewer validates results

## Current Status
- UI: Prototype (Streamlit)
- Backend: FastAPI
- Data: Synthetic / public documents only

## Planned Enhancements
- Visual clause highlighting
- Reviewer feedback storage
- Multi-question sessions
